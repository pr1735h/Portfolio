Welcome to my portfolio website!

I use this space to show off my skills and explain my thought process. I found that weird wavy font (Wavefont) on google fonts. It looks like an equalizer to me so I wrote some JavaScript functions. The first one declares a variable with a list of all digits, upper and lowercase letters, then it looks at the length of the original text and randomly reassigns each character with a character from the list. Another function changes the values of the "YELA" attribute of the font. Both functions use math.random and are called every 0.5 seconds
